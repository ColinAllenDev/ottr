# Wayland Notes

## Dependencies Providing Additional Functionality
- KMS/DRM (Kernel Mode Settings / Direct Rendering Manager)
  - Interfaces with GPUs to render to displays
- Input Library: libinput
  - A library for handling the hardware portions of input devices that make up Wayland seats.
- X11 Compatibiliy Layer: XWayland
  - A compatability layer for running X11 applications in Wayland.
- Keyboard Descriptions and Events: xkbcommon
